Floppy Bird

โครงการนี้มีวัตถุประสงค์เพื่อศึกษาการพัฒนาเกมโดยใช้ภาษา C โดยไม่พึ่งเกมเอนจินสำเร็จรูป และเพื่อทดลองนำโค้ด C มาพอร์ตขึ้นเว็บผ่านเทคโนโลยี WebAssembly เกมที่พัฒนาคือ “Floppy Bird” ซึ่งได้รับแรงบันดาลใจจากเกม Flappy Bird

Project Details

Library ที่ใช้

  • SDL2 — สำหรับสร้างหน้าต่างเกมและ render ต่างๆ
  • SDL2_image — สำหรับโหลดรูปภาพ
  • SDL2_mixer — สำหรับ sound effect และเพลงประกอบ
  • SDL2_ttf — สำหรับโหลดฟอนท์

การทำงานภายในเกม

  • Bird: สร้างสี่เหลี่ยมผืนผ้าพร้อมโหลดภาพ Bird.png
  • Gravity & Jump Mechanic: ขยับรูปนกลงตามค่า Gravity ทุกๆ เฟรม และเมื่อผู้เล่นกด Spacebar จะลบค่าตาม FlapStrength ทำให้นกเด้งขึ้น
  • Dash Mechanic: เมื่อผู้เล่นกด Shift นกจะหยุดความเร็วชั่วขณะ (ลอยอยู่กับที่) และเปลี่ยนภาพเป็นรูปพุ่ง เพิ่มความเร็วท่อด้วย Dash speed ให้เหมือนนกบินเร็วขึ้น
  • Collision Check: ตรวจสอบพิกัดและขนาดของ A เทียบกับ B หากทับกันถือว่าชน
  • Pipe: สร้างท่อคู่ห่างกันตามค่า Pipe gap = 250px พร้อมสี่เหลี่ยมล่องหนตรวจผ่าน เมื่อผ่านได้ คะแนน +1 ท่อเคลื่อนที่ซ้ายตาม Pipe speed สร้างภาพลวงตาการบิน
  • Pipe Spawn: ทุกเฟรม Pipe timer จะเพิ่มขึ้น เมื่อครบค่า (80) จะหาท่อว่างใน MAX_PIPE และเปิดใช้งานใหม่ (active = 1, scored = 0)

อุปสรรคและข้อจำกัด

ในระหว่างพัฒนาโครงการ มีอุปสรรคเรื่องการทำฐานข้อมูลสำหรับ leaderboard ซึ่งไม่สามารถทำได้ตามแผนเดิม จึงเลือกที่จะเลื่อนฟีเจอร์นี้ไปก่อน ปัจจุบันตัวเกมสามารถเล่นได้แล้ว แต่ยังไม่มีระบบ leaderboard สำหรับเรียงลำดับคะแนนผู้เล่นสูงสุด (Top 10) ตามที่วางแผนไว้ตั้งแต่แรก
