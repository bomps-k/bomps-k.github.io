Floppy Bird

A modern take on the classic Flappy Bird game built with HTML5, CSS3, and JavaScript

Project Details

Description

Floppy Bird is a browser-based game inspired by the popular Flappy Bird. The project demonstrates core JavaScript concepts including:

  • Game loop implementation using requestAnimationFrame
  • Collision detection algorithms
  • Event handling for user input
  • DOM manipulation for game rendering
  • Score tracking and game state management
